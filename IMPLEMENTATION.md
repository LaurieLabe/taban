# 她伴 · 代码实施计划

> 版本：v1.6 | 2026-07-19

---

## 整体原则

- **后端先行，前端跟进** — 后端 API 先跑通，前端再对接
- **每步 commit** — 每完成一个模块就存档，回归点清晰
- **每阶段验证** — 阶段末尾有验证清单，逐条打勾确认
- **先骨架后血肉** — 先搭能跑通的框架，再填具体逻辑
